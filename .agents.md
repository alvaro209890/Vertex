# Vertex

DeepSeek-native proxy for Claude Code CLI. Routes Anthropic Messages API traffic exclusively to DeepSeek models via `api.deepseek.com/anthropic`.

## Quick Start

```bash
# Dependencies: Claude Code CLI, uv, Python 3.12+
uv sync
cp .env.example .env
# Edit .env: set DEEPSEEK_API_KEY
uv run uvicorn server:app --host 0.0.0.0 --port 8082
```

Then connect Claude Code:
```bash
ANTHROPIC_AUTH_TOKEN="freecc" ANTHROPIC_BASE_URL="http://localhost:8082" claude
```

## Architecture

- **server.py** — ASGI entrypoint (uvicorn)
- **api/app.py** — FastAPI app factory with lifespan, error handlers
- **api/routes.py** — Route handlers (`/v1/messages`, `/v1/models`, `/health`, etc.)
- **config/settings.py** — Pydantic settings from `.env` / environment
- **config/provider_catalog.py** — Provider descriptors (DeepSeek, NIM, OpenRouter, etc.)
- **providers/registry.py** — Provider factory + runtime cache
- **providers/deepseek/** — DeepSeek Anthropic-native transport
- **core/anthropic/** — Shared Anthropic protocol utilities
- **cli/** — CLI session management, entrypoints
- **messaging/** — Optional Discord/Telegram bot adapters

## Models

| Model ID | Description |
|---|---|
| `deepseek/deepseek-v4-pro` | Full reasoning (thinking mode) |
| `deepseek/deepseek-v4-flash` | Fast responses (non-thinking) |
| `deepseek/deepseek-chat` | Legacy (deprecated 2026/07/24) |
| `deepseek/deepseek-reasoner` | Legacy (deprecated 2026/07/24) |

## Configuration (`.env`)

| Variable | Description |
|---|---|
| `DEEPSEEK_API_KEY` | DeepSeek API key (required) |
| `MODEL` | Default model |
| `MODEL_OPUS` | Override for Opus-tier requests |
| `MODEL_SONNET` | Override for Sonnet-tier requests |
| `MODEL_HAIKU` | Override for Haiku-tier requests |
| `ANTHROPIC_AUTH_TOKEN` | Proxy auth token |
| `MESSAGING_PLATFORM` | "telegram", "discord", or "none" |
| `ENABLE_MODEL_THINKING` | Enable thinking/reasoning blocks |
| `PORT` | Server port (default 8082) |

## API Endpoints

- `POST /v1/messages` — Stream a message (Anthropic Messages API)
- `POST /v1/messages/count_tokens` — Count tokens
- `GET /v1/models` — List available models
- `GET /health` — Health check

## Clients

- **Claude Code CLI**: `ANTHROPIC_AUTH_TOKEN="freecc" ANTHROPIC_BASE_URL="http://localhost:8082" claude`
- **VS Code**: Set `claudeCode.environmentVariables` with `ANTHROPIC_BASE_URL` and `ANTHROPIC_AUTH_TOKEN`
- **JetBrains**: Edit `~/.jetbrains/acp.json` or Windows equivalent

## Commands

- `vertex` — Start the proxy server
- `vertex-init` — Scaffold default config at `~/.config/vertex/.env`
- `claude-pick` — Interactive model picker (requires `fzf`)
